 📊 Sales Analysis Dashboard
 📌 Overview
This project showcases an interactive Sales Analysis Dashboard designed to provide actionable insights into business performance. The dashboard analyzes key metrics such as revenue, cost of goods sold (COGS), profit, and customer distribution.

It enables users to explore trends, compare performance across regions and product categories, and make informed, data-driven decisions.

---

🎯 Objectives
- Analyze overall revenue, profit, and cost efficiency
- Evaluate sales representative performance
- Identify high-performing products
- Track monthly profit trends
- Compare regional cost distribution
- Enable dynamic filtering for deeper insights

---

 📈 Key Features
- KPI Metrics
  - Total Revenue
  - Total COGS
  - Total Profit
  - Total Customers

- Visual Insights
  - Sales Rep by Revenue (Bar Chart)
  - Product by Profit (Line Chart)
  - City by COGS (Pie Chart)
  - Monthly Profit Trends (Column Chart)

- Interactive Filters
  - Region (East, North, South, West)
  - Product Category (Electronics, Furniture, Home Appliances)

---

 🛠️ Tools & Technologies
- Microsoft Excel
  - Pivot Tables
  - Pivot Charts
  - Slicers
  - Data Cleaning & Transformation
